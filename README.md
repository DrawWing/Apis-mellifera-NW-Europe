# Analysis of honey bee wing shape in northwestern Europe

The file [Apis-mellifera-NW-Europe.html](https://drawwing.github.io/Apis-mellifera-NW-Europe/Apis-mellifera-NW-Europe.html) contains an R Markdown document that analyzes the wing shape of honey bees (*Apis mellifera*) from northwestern Europe.\
